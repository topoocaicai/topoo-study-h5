01 · BRAND PHILOSOPHY
三个核心理念
拓圈筛选的是精神状态，不是职业头衔。这三条是入场门槛，不是口号。
01

长期主义

不急功近利，愿意花时间打磨自己和产品，相信慢就是快。拓圈不是流量收割机，是需要时间才能见效的生态。

02

探索精神

知道自己是谁，或者正在寻找自己是谁。拓圈给你一个空间自己去发现，没有人告诉你该往哪走。

同频共振

相信碰撞的力量，愿意跟一群人一起走。不是一个人抱着手机单打独斗，是找到真正同频的伙伴。

04 · CREATOR TYPES
创造者五行属性
五种创造者类型，五行相生，构成完整的生态循环。一个健康星球需要五行均衡。
💧
水
内容型
无形流到任何角落。内容就是水，渗透种草，流动即传播。
短视频 / KOL / IP / 博主
🌱
木
产品型
从种子到果实，需要水（内容）和阳光（流量）滋养生长。
实体商品 / 手作 / 线上产品
🔥
火
门店型
门店是火塘，人聚集的地方，烟火气，有固定位置有真实温度。
咖啡店 / 茶叶店 / 手作店
🌍
土
活动型
活动就是土，承载万物，是所有创造类型的场地和地基。
展览 / 市集 / 工作坊 / 快闪
⚙️
金
服务型
精准切割需求，锻造精钢，高价值硬通货，让其他人更强。
设计 / 策划 / 咨询 / 创意
水·内容 → 滋养 → 木·产品 → 带动 → 火·门店 → 催生 → 土·活动 → 锻炼 → 金·服务 → 产出 → 水·内容 （循环）

06 · NUMBER ECONOMY
号段经济模型
号不是消费，是投资。越早进来，号越值钱。
01 · SCARCITY

号码稀缺性

1-10号预留拍卖，号码越靠前越值钱。类似 QQ靓号/车牌号逻辑，物以稀为贵，前排号天然有价值。

02 · ACCUMULATION

权重积累

越早进来，内容积累越多、交易量越大、等级越高。后续新漫游者进来最先看到高权重创造者——这就是流量分配权。

03 · TRADEABLE

号可交易

不想做了可以卖号，前期2000进的号，养好后卖1万2万都可以。权重跟着号走，买家买的是积累的一切。

07 · GAMIFICATION
游戏化底层战略
参考《无尽冬日》联盟机制。游戏化做内核，商业化做外衣。
⬆️

资源永远不够·逐步解锁

升级目标永远有下一步，不一步到位。让用户始终有奔头，阶段性成就感不断输出。

↔ 对标：圈科技升级，内容权重 +10%
🤝

联盟绑定·社交关系网

社交关系越织越密，离开成本越来越高。从陌生人到盟友到真实朋友，这个过程是拓圈的核心价值。

↔ 对标：圈月度挑战赛，集体积分冲Top3
📊

同侪压力·看见彼此成长

能看到同期的人谁在进步，对比本身就是驱动力。不是攀比，是激励——你的成长被人看见，别人的进步激励你。

↔ 对标：季度黑马榜，城市圈对抗
⏰

限时活动·阶段性冲刺

月度挑战赛/季度黑马榜/限时资源加成。参加是锦上添花不是保命必需，有点紧迫感但不制造焦虑。
