# Day 1 视频脚本：「我要在南京8天拉1000人」

## 基本信息

| 项目 | 内容 |
|------|------|
| 平台 | 抖音（主）+ 小红书（图文版同步） |
| 时长 | 60-75秒 |
| 出镜 | 不出镜，PPT翻页+实拍穿插 |
| 视觉风格 | 黑白灰，干净克制，数据驱动 |
| 拍摄工具 | PPT动画 + 手机实拍镜头（手机随手拍即可） |

---

## PPT视觉规范

- **背景色**：深黑 #1A1A1A 为主，白字 #FFFFFF，强调用亮灰 #A0A0A0
- **字体**：思源黑体/苹方，无衬线，粗体标题+细体正文
- **动效**：淡入淡出为主，不用花哨动画，每一页停留5-8秒
- **数据展示**：数字要大，单独成行，有冲击力

---

## 分镜脚本（60-75秒）

### 第一段 | 钩子（0-5秒）

**PPT画面：**
- 黑屏，白色大字逐行淡入——

```
南京
8天
1000人
```

- 三个数字依次出现，每行间隔0.8秒
- 最后"1000人"字号最大，停留1秒

**配音：**
"大家好，我是苏菜菜，拓圈Topoo发起者。"

---

### 第二段 | 交代背景（5-18秒）

**PPT画面：**
- 黑底白字，标题淡入——"我要做一件大概率会被打脸的事。"
- 停留2秒后，画面切换——

**穿插镜头①（2秒）：** 手机屏幕录屏——空白的报名页面/活动行页面（真实截图）

**PPT画面：**
- 新页面淡入，信息逐条出现——

```
5月10日 — 5月17日
南京·国创园
6场线下培训
讲师：张明军（百万粉科技博主）
```

**穿插镜头②（2秒）：** 张明军账号主页截图/课程海报（快速闪过）

**配音：**
"5月10号到17号，8天时间，6场线下活动。

我要在南京，拉满1000个人到场。

不是线上看直播，是真人坐在椅子上，待一整天。

讲什么？短视频加AI，讲师张明军，百万粉丝科技博主。"

---

### 第三段 | 制造争议（18-35秒）

**PPT画面：**
- 一页简洁的数据展示——

```
南京常住人口：940万
1000人 = 万分之一
```

- 停留2秒，然后画面切——

**穿插镜头③（3秒）：** 实拍——空荡荡的场地/椅子还没摆好的样子（国创园或者其他空场地）

**PPT画面：**
- 黑底白字，大号——

```
但做过线下活动的人都知道

让一个人——
看到信息 → 付199块 → 周末爬起来出门
→ 坐地铁到场 → 坐8个小时

每一步，都在掉人。
```

- 文字逐行淡入，"掉人"二字用红色 #FF4444 或亮灰 #CCCCCC 强调

**配音：**
"你可能觉得，南京940万人，1000人算什么？

但做过线下活动的人都知道——

让一个人从看到你的信息，到付199块钱，到周末早上爬起来出门，到坐在椅子上待8个小时——

这中间每一步，都在掉人。"

---

### 第四段 | 算账（35-50秒）

**PPT画面：**
- 白底黑字（或黑底白字），大号数字，居中——

```
6场 × 167人 = 1002人
```

- 数字有动画效果（比如从0跳到167、从0跳到1002）

**PPT画面切换：**
- 黑底，白色大号数字——

```
距首场：14天
当前报名人数：
```

- 停顿1秒——

```
0
```

- "0"字单独出现，字号超大，停2秒

**穿插镜头④（2秒）：** 实拍——你打开手机看报名后台的画面积攒（手部特写，不露脸）

**配音：**
"6场活动，平均每场167人，刚好1002。

第一场5月10号，还有14天。

现在报名人数——

零。

从今天开始，我每天更新真实数据。"

---

### 第五段 | CTA+收尾（50-65秒）

**PPT画面：**
- 黑底白字，简洁——

```
做到了 → 算本事
没做到 → 复盘素材
反正，不会无聊。
```

- 逐行出现

**PPT画面切换：**
- 信息页——

```
南京·国创园
5月10日-17日
¥199 / 人（全天6大模块）
线下授课 · 张明军
```

- 最后加一行闪烁——

```
点链接，报名。
```

**穿插镜头⑤（2秒）：** 实拍——场地一角/大门/logo墙（让场地有真实感）

**配音：**
"做到了算本事，没做到这个系列本身就是最好的复盘素材。

如果你在南京，做短视频或者想做短视频，点链接。

199块，一天，6大模块，从早到晚。"

---

### 第六段 | 尾帧（65-70秒）

**PPT画面：**
- 黑屏，固定信息——

```
Day 1 / 22
━━━━━━━━━━━━━
当前报名：0人
目标：1000人
距首场：14天
━━━━━━━━━━━
拓圈 Topoo
让小而美被世界看见
```

- "Day 1 / 22"字体最大，其余信息整齐排列
- 停留3-5秒，让观众截图/记住

**配音：** 无，纯尾帧展示
