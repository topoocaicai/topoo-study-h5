## 一、项目定位

### 1.1 拓圈是什么

拓圈不是一个公司，是一个**集群/联盟**的概念，是整合多方资源和能力的**平台品牌**。

核心理念：**让小而美被看见。**

### 1.2 终极愿景——自生长

核心公式：**定位准确 + 规则设定 + 逻辑关系理清 = 自生长**

- 拓圈在自生长，帮每个人自生长
- 类比DAO组织：共建共创共治共享，但有核心不独裁
- 先集中后分布——创始人拍板定规则，等生态成熟再放权
- 先跑通最小闭环，自生长作为愿景不作为对外主打

对外叙事：**"我们把自生长跑通了，现在帮你也跑通。"**

---

## 二、组织架构——集群概念

拓圈是多方资源整合的统一平台品牌，各主体各司其职：

- **菜菜本人**：拓圈核心发起人和灵魂人物，串联所有资源
- **南京璞玉网络技术有限公司**：合伙人公司，承办抖音电商相关活动（如618策略宣贯会）
- **南京振业**：负责大学生直播大赛等高校方向活动

## 三、底层战略——游戏化思维

### 3.1 为什么是游戏化

- 用游戏的思维做拓圈平台，每个人是"角色点"，有技能树、经验值、升级路径
- 即时反馈+多巴胺驱动：学习/工作难坚持是因为反馈链路太长，游戏上瘾是因为即时反馈
- 学中玩，玩中学：工作不是苦大仇深，要符合年轻人心态，在玩中实现价值观
- 菜菜本人是游戏玩家基因，用游戏思维做项目是底色

### 3.2 游戏化原则

- 游戏化做内核，商业化做外衣——机制是游戏的，体验是专业的，别太游戏化别让人觉得弱智
- 每一步赋能都有即时反馈设计，但外现专业克制

## 四、角色体系（四层）

### 4.1 总览

四层角色结构：

- **创造者 Creator**：做产品/内容/IP，变现得货币
- **工作者 Worker**：辅助创造者，碳基得货币+硅基用token
- **链接者 Linker**：搭桥/推广/撮合
- **探索者 Seeker**：新手村，蓄水池

### 4.2 创造者（Creator）——核心资产

#### 本质

创造者只有一个本质：**他有一个东西要卖。**

#### 关键公式

**灵魂 → 外挂大脑（AI） → 拓圈（落地） → 共鸣**

- 创造者拥有灵魂（唯一不可替代）
- AI是外挂大脑（天马行空）
- 拓圈帮想法在现实中落地站稳
- AI是工具不是裁判，AI是讨好型人格，不能替你做判断
- 共鸣是因，交换是果，没有共鸣就没有交换

#### 三类型赋能路径

**（1）产品型创造者——五步法**

1. 诊断定位
2. 内容打磨
3. 渠道铺设
4. 验证调优
5. IP放大

- 核心：产品型是拓圈的根据地，有实际运营经验和团队
- 产品技能点体系：诊断打分 → 找到天然爆点 → 集中火力拉爆 → 出圈再补其他

**（2）内容型创造者——五步法**

1. 定位诊断
2. 内容+选品一体化
3. 内容制作与发布
4. 数据验证与优化
5. 商业变现与IP放大

- 核心：不是教做内容，而是让内容有东西可卖
- 拓圈壁垒：抖音电商官方服务商身份+内容选品一体化+跨界内容搭建+前期流量扶持+场域内流量互转+广告合作/带货货源/课程推广全链路对接
- 工作者（碳基+硅基）全程辅助，产品型的能力可以复用到内容型上
- 三种创造者互相赋能，飞轮效应

**（3）服务型创造者——五步法**

1. 能力包装
2. 服务产品化
3. 生态对接
4. 信任建设
5. 能力放大

- 定义：以专业技能和创意能力为核心的服务提供者（设计/策划/营销/咨询/创意类）
- 服务型创造者 = 自雇型为自己干（平台上的独立商户）
- 碳基工作者 = 为平台打工按平台规则来（平台工作人员）
- 壁垒主要靠内部生态：三种创造者互相成为客户

#### 创造者定位坐标系

创造者三维度（产品/内容/服务）× 四阶段（有→见→信→续），二维坐标系定位。

### 4.3 工作者（Worker）——专业支撑

#### 碳基工作者

四个专业方向：
- 品牌运营
- 创意执行
- 商业对接
- 数据策略

成长路径四级（自裂变机制）：
- 见习 → 正式 → 资深 → 导师级

#### 硅基工作者（AI）

五大AI模块：
- 内容生产
- 数据分析
- 诊断咨询
- 运营自动化
- 学习成长

#### 碳基+硅基协作逻辑

- AI做70%基础工作，人做30%关键判断
- 产品篇+内容篇能力可复用到工作者培训

### 4.4 链接者（Linker）——连接器

#### 核心定位

创造者是节点，Linker是节点之间的线。

创造者自己发光，Linker让别人发光。他不生产，他连接。

#### 核心能力

1. **撮合对接**：知道谁需要谁，判断谁和谁搭，主动搭桥
2. **推广传播**：拓圈的外部触角，拉新、推广活动、传播理念
3. **活动组织**：线下聚会、沙龙、私董会、路演
4. **资源整合**：对接外部资源（政府、品牌方、媒体、其他平台）

#### 经济模型

- 前期免费，但设转正考核
- 转正条件：3个月内成功撮合3单或拉新10人
- 考核通过后正式成为Linker，可收低额年费或保证金增加沉没成本
- 考核不通过回到普通用户身份

### 4.5 探索者（Seeker）——蓄水池

#### 定位

拓圈的漏斗入口，待孵化的创造者储备池。就像粉丝池——人数越多，对创造者的吸引力越强。

#### 升级路径（三条线可混搭）

1. **任务线（免费攒经验）**：完成简单任务获得经验值
2. **付费线（花钱加速）**：付费解锁加速成长
3. **积分线（日常活跃）**：日常活跃攒积分

#### 毕业机制

- 新手村毕业时做"角色意向测试"
- 系统推荐+用户自己决定选什么角色

## 五、经济模型

### 5.1 各角色收费模型

| 角色 | 收费方式 | 说明 |
|------|----------|------|
| Creator（种子期） | 免会员费（终身）+ 保证金 | 保证金金额待定，退还条件与活跃度/贡献度挂钩 |
| Creator（正式期） | 以完成任务形式免费进入 | 游戏化上瘾机制，不设会员费门槛 |
| Worker | 培训费（前期可免） | 不是雇佣关系是赋能关系，按件计费实时结算 |
| Seeker | 前期免费（完成任务） | 后期可收月卡/年费 |
| Linker | 前期免费+转正考核 | 转正后可收低额年费或保证金 |

### 5.2 成本结构——交易内嵌式

**核心逻辑：不承担固定人力成本，所有人员成本内嵌在每笔交易里。**

- 碳基工作者：按件计费，实时结算，从创造者付的服务费里直接分成
- Linker：按结果付费，拉来的人/撮合的订单直接从佣金里分成
- 每笔收入进来之前，成本已经拆好：平台抽成（拓圈利润）+ 工作者结算 + Linker佣金
- 平台拿到手的才是净利润

### 5.3 商业模型底层逻辑——概率筛选+幸存者放大

1. 大量种子进来（基数大）
2. 平台赋能（五步法+工作者+AI辅助，提高每个人成功的概率）
3. 大部分人没跑出来（正常，这就是概率）
4. 跑出来的那几个放大成案例（案例是真的）
5. 案例吸引更多人进来（飞轮启动）
6. 没跑出来的人退还保证金，但他们在过程中贡献了活跃度、内容尝试和生态养分

**平台赚的是跑出来的创造者的持续交易抽成。**

赋能体系把成功率从1%拉到4-5%，这2%-3%的增量就是拓圈最大价值。

## 六、南京城市圈——第一年打样

### 6.1 为什么是南京

- 菜菜在南京，能直接管、亲自跟
- 南京消费力不错，文创氛围好
- 已有本地资源和人脉
- 高校资源丰富（拉新渠道）
- 城市不大不小，适合做深做透

### 6.2 线下根据地——国创园

**基本信息：**
- 运营方：金基集团（1992年成立）
- 前身：南京第二机床厂（1896年江南铸造银元制钱总局），百年工业遗存
- 占地7.1万㎡，老厂房面积5.9万㎡
- 在园企业133家，在园人数超3000人
- 年产值超20亿，年税收约1.5亿

**产业定位：**
- 文化创意+设计服务+科技创新三大主导产业
- 文化企业占比60%以上
- 代表企业：洛可可、兴华设计院、金锋团队等

**空间特色：**
- 23BOX/25BOX复合文化艺术空间（适合活动路演、艺术展览）
- 物色市集（金基自创品牌，南京新青年文化聚集地）
- 江南造币博物馆、工业遗存主题公园八景

**与拓圈的匹配度：**
- 工业遗存+文创基因，与拓圈"文化+创意"底色天然契合
- 本身就是创意产业聚集地，有潜在创造者和服务型创造者
- 园区场地费用算宠物项目，不占拓圈成本

## 八、COSEE星球——拓圈前身参考

COSEE是菜菜两年前的项目，拓圈在COSEE基础上升级（游戏化+AI+更广人群）。

### 8.1 COSEE基本信息

- COSEE = 链接你我，可见未来
- 去中心化、女性为主、链接为目的

### 8.2 三角色体系

- 探索者Seeker（基石）→ 创造者Creator（深度参与）→ 链接者Linker（桥梁）

- 小组Team（兴趣3人起步）
- 项目Project（共建）
- COSEE X（私域圈）

### 8.4 跨界公式

70%A × 70%B × 70%C > 90%A

不同基因杂糅才能突变新事物。

### 8.5 COSEE活动体系

COSEE TALK、私董会、训练营、企业走访、读书会、运动圈等

### 8.6 COSEE收费

- 探索者500元/年，创造者2000元/年，链接者2000元/年
- 种子限量免费+守护金
