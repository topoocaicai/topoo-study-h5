<summary><strong>一、为什么要成为创造者？</strong></summary>

### 你现在的问题

你做了产品，拍了图，写了文案，上了架。然后——

你发现流量是买来的，客户是算出来的，复购是靠打折换来的。你不停地产出，但越来越觉得：这不是我想要的状态。

**核心问题不是你做得不够好，是你的东西没有遇到"对的人"。**

算法给你流量，但算法不了解你。它把你的产品推给1000个人，其中可能只有3个真的需要。剩下997个划走，你的内容就"表现不好"，算法就不推了。

这是喂养逻辑——你在被喂养流量，同时你也在喂养算法。

### 拓圈里的创造者是什么？

不是"商家"，不是"博主"，不是"供应商"。

创造者是**在拓圈里认真做东西、并把东西拿出来和人碰撞的人**。

你有一个产品、一项服务、一个内容方向——你把它放进拓圈，让人碰撞到它。有人路过你的门牌，有人被你的故事打动，有人点亮你，有人把你的东西放进他的星空。

这些不是"转化"，是**碰撞**。

碰撞产生的不是一笔订单，是一个**真实的关系**。

### 创造者和探索者的区别

| | 探索者 | 创造者 |
|--|--|--|
| 核心行为 | 主动探索，碰撞别人 | 做出东西，被别人碰撞 |
| 空间功能 | 个人成长记录 | 产品展示 + 委托接单 |
| 收益来源 | 分配 + 号段交易 | 产品销售 + 服务委托 + 分配 |
| 入场门槛 | ¥365终身 / ¥599年 | 申请制 + 保证金 |

创造者不是"升级版探索者"——是**另一种角色**，做的事不一样，收益方式不一样，在拓圈里的位置也不一样。

<details>
<summary><strong>二、创造者能做什么？</strong></summary>

### 你的空间（创造者的"小家"升级版）

探索者的空间是用来"记录自己"的。创造者的空间，除了记录自己，还有一个核心功能：**展示你的东西**。

**空间分层：**

```
第1层：门牌（别人路过就能看到）
第2层：展示区（别人进入空间后看到的产品/服务列表）
第3层：故事区（你为什么做这个东西）
第4层：委托区（别人可以直接在这里给你发委托）
```

**门牌（外部门牌）**
和探索者一样，创造者也挂3D门牌。但创造者的门牌上多了一行字：
> "此人可以做：[你的服务类型]"
> 比如："此人可以做：品牌设计 / 包装设计"

别人路过看到这行字，就知道可以给你发委托。

**展示区（空间内部）**
最多展示12个产品/服务。每个产品有：
- 封面图
- 名称
- 一句话描述
- 价格（可选，也可以写"私信询价"）
- 被点亮次数（社交证明）

**故事区**
你为什么做这个产品？你遇到过什么问题？你花了多长时间打磨？

这不是"品牌故事"那种包装文案。是**你真实的思考过程**。拓圈里有人会因为你的故事买单，不是因为你的产品有多完美。

**委托区**
别人可以直接在空间里给你发委托："我想做一个XX，你能帮我吗？"
你收到委托通知，可以选择接或不接。
接了之后，系统生成"铸造成果"，积累进你的专业档案。

---

### 拓圈前期扶持计划（我们帮你起步）

> 你认真做东西，我们认真帮你把东西放对地方。

创造者最难的不是"做东西"，是**刚开始没人看见、不知道怎么展示、不知道有什么玩法**。

拓圈针对前期创造者，提供三项扶持——

**① 空间搭建 1 对 1 指导**
你不知道门牌上写什么、展示区放什么图、故事区怎么写？
拓圈团队会和你一起打磨——不是帮你"包装"，是帮你把**真实的东西整理得让人能碰撞到**。

**② 新玩法共创**
你有一个新想法（比如：我想做一个「旧物修复」的委托类型、我想做每周限时闪购），但不知道怎么实现？
拓圈产品团队可以和你一起把它做出来——**你的玩法，可能变成拓圈所有创造者都能用的功能**。

**③ 你的空间，也可以有"拓圈同款"**
你的空间和橱窗里，可以应用拓圈同一套机制——碰撞、点亮、灵魂画像、五行场域……
我们会提供工具和模板，让你在自己的空间里也能跑这套系统，而不是一个静止的展示页。

> 简单说：你不是进来之后自己摸索，**拓圈和你一起把这件事做起来**。

---

### 产品展示与销售

**创造者在拓圈里可以卖东西吗？**
可以。这是创造者最重要的功能之一。

**产品上架：**
- 最多同时展示12个产品
- 每个产品有独立页面，包含：图、描述、价格、库存、"被点亮次数"
- 支持"私密上架"：只让碰撞到你的人看到（不公开到广场）

**交易方式：**
- 拓圈不收交易佣金（种子期政策）
- 交易通过微信小程序完成（拓圈是门面，小程序是交易场）
- 创造者可以设置"号段持有者专享价"（鼓励别人来碰撞你）

**产品被点亮：**
和探索者的空间被点亮类似——别人买了你的产品、或深度被你的产品故事打动，可以"点亮"你的产品。
点亮后：
- 你的产品页面会显示"已被XX人点亮"
- 产品在主搜索/推荐里的权重上升
- 你获得一个"产品光点"，累积进灵魂画像

---

### 委托接单系统

**什么是委托？**
别人需要你的专业能力，通过你的门牌/空间直接给你发任务。

**委托流程：**
1. 对方在你的空间里点击"发委托"
2. 填写需求描述、预算范围、时间要求
3. 你收到通知，选择"接"或"不接"
4. 接了之后，系统生成"铸造中"状态
5. 完成后，对方确认，系统生成"铸造成果"归档

**委托的专业档案：**
每次完成委托，系统自动生成一条记录：
> "20XX年X月，完成了[委托人]的[项目类型]，耗时X天"

这些记录累积起来，就是你的**专业能力档案**。不是评分，是事实记录。别人来委托你之前，会看这个档案。

**金型创造者的特权：**
金型（服务型）创造者的门牌上自带"可委托"标记，系统会优先把委托需求推荐给你。

---

### 收益模式（创造者怎么赚钱）

创造者有三个收益来源——

**① 产品销售**
直接卖产品，收入归你。拓圈种子期不收交易佣金。

**② 服务委托**
别人给你发委托，你接单完成后收费。定价你自己定，平台不干预。

**③ 收益分配**
拓圈整体有收益后（广告、企业服务、活动收入等），会拿出一部分分配给所有角色。**创造者的分配权重比探索者高**。

创造者的分配权重计算方式：
```
基础权重（和探索者相同）= 等级档位
额外权重 = 你的产品被点亮次数 + 你完成的委托数 + 你的相生合作次数
```

简单说：**你做得越多、和别人碰撞得越多，分配越多**。

**④ 号段交易（和探索者相同，但创造者的号更值钱）**
你的号段是**资产，不是消费**。

创造者号段的价值 = 等级积累 + 产品资产 + 专业档案 + 灵魂画像完整度 + 五行属性成长记录。

**这个号未来可以交易**——越早入场，积累越多，号段价值越高。
你不是"买了一个创造者资格"，是"在拓圈里经营了一个会增值的资产"。

<summary><strong>四、如何成为创造者？</strong></summary>

### 申请制（不是谁都能进）

创造者不是买个会员就自动获得的身份。你需要**申请**，拓圈审核通过后才能真正成为创造者。

**为什么？**
因为创造者的门牌会出现在街区地图上，别人会看到你、委托你、买你的产品。如果随便什么人都能成为创造者，拓圈的信任基础就没了。

**申请条件：**
1. 已经有成型的产品/服务/内容方向（不是"我想做"，是"我已经在做"）
2. 愿意接受平台的碰撞机制（被点亮、被留光点、被写进别人的笔记本）
3. 同意不造假、不包装、不造神（违反直接清退，保证金不退）

### 种子期类目保护机制

原则上，**种子期一个细分类目只选1-3家**。

为什么？
因为拓圈不想做成"万物商城"，我们想让早期进来的创造者有足够的空间慢慢生长，而不是一进来就和白热化的竞品卷价格。

如果你做的类目已经满额了，可以选择：
1. 等待下一批次开放
2. 换成另一个你也在做的类目
3. 先以探索者身份入场，碰撞一段时间再申请

### 保证金制度

| 阶段 | 金额 | 说明 |
|---|---|---|
| 种子期 | ¥2000（可退） | 前100个创造者专享，可退 |
| 正式期 | ¥2999 + ¥5000保证金 | 常态门槛 |

**关于保证金退出：**
- 如果不想继续做了，可以申请退保证金
- **如果有用户已经购买你的产品或委托你服务，需要完成所有已产生的交付后再退出**
- 没有投诉记录、账号无违规，审核通过后7个工作日内退回

**为什么要保证金？**
不是割韭菜，是**建立信任门槛**。
创造者是要被别人委托、被别人买产品的人，如果没有任何门槛，拓圈就变成了一个"人人都可以来卖东西"的平台，信任就没了。

**你的号，是很值钱的资产：**
创造者的号段 = 等级积累 + 产品资产 + 专业档案 + 灵魂画像完整度 + 五行属性成长记录。
这个号未来**可以交易**——越早入场，积累越高，号段价值越高。
不是消费，是资产。

### 申请流程

```
提交申请 → 平台审核（3个工作日）→ 通过 → 缴保证金 → 开通创造者权限
```

审核标准：
- 你有真实的产品/服务/内容，不是空壳
- 你的东西和拓圈的价值观对齐（小而美，不造神）
- 你的表达方式是真实的，不是包装出来的

<summary><strong>五、五行力量体系（创造者的"天赋树"）</strong></summary>

> 这是一套完整的系统，不是装饰。你可以慢慢看，不着急一开始全懂。

### 核心概念：五行不是你选的，是你长出来的

你注册时不用勾选"我是火型还是水型"。系统观察你做了什么，自动判断你的主导力量。

**五种力量：**

| 五行 | 名字 | 本质 | 你的天赋倾向 |
|:---:|:---:|:---|:---|
| 🔥 火 | **聚集力** | 让人聚过来 | 天生擅长做空间/门店，人在你这里愿意停留 |
| 💧 水 | **渗透力** | 内容流到远处 | 天生擅长做内容，陌生人会因为你的内容走进来 |
| 🌱 木 | **生长力** | 想法变实体 | 天生擅长做产品，从0到1把东西做出来 |
| ⚔️ 金 | **打磨力** | 混乱变秩序 | 天生擅长做服务，别人有问题你就能解决 |
| 🏔️ 土 | **承载力** | 让所有人有地方待 | 天生擅长做活动/空间，把人聚在一起 |

---

### 五行场域效应（系统加成）

每种力量会给你带来三个实际的系统加成——

**🔥 火（聚集力）**
1. 路过加成：别人地图上你的门牌会微微"暖"一点，更容易被注意到
2. 停留buff：有人进入你空间后，停留时间统计权重×1.5（影响灵魂画像的"深度"维度）
3. 转化加成：火型创造者获得的碰撞，转化为"深度连接"的概率更高

**💧 水（渗透力）**
1. 流动加成：你的内容出现在"碰撞推荐"里的权重更高
2. 共鸣加速：被人点亮后，对方获得额外光点×1（鼓励继续传播）
3. 广度标记：灵魂画像里会标注"此人擅长广度连接"

**🌱 木（生长力）**
1. 实物锚点：门牌上可以挂"产品预览"，别人路过直接看到你做了什么
2. 生长追踪：系统记录"从种子到果实"全过程，产品迭代自动归档进灵魂画像
3. 交易溢价：号段交易时，木型创造者的产品资产会折算成额外溢价系数

**⚔️ 金（打磨力）**
1. 信赖标记：门牌上有"可委托"标记，别人可以直接给你发任务
2. 案例沉淀：每次完成委托，系统自动生成"铸造成果"，累积成你的专业能力档案
3. 定价自由：金型创造者的服务定价不被平台限价，完全市场决定

**🏔️ 土（承载力）**
1. 场地优先：发起活动时，系统优先推荐你的空间作为聚集点
2. 人数buff：你发起的活动，初始曝光权重×2
3. 跨属性桥梁：你组织活动时，参与的创造者无论什么属性，合作权重都+0.5

### 五行相生（合作加成机制）

相生不是玄学，是**鼓励跨类型合作的系统机制**。

相生方向：水→木→火→土→金→水

**具体怎么触发？**
- 木型创造者做了新产品
- 水型创造者的内容帮它传播了
- → 系统检测到这次"相生行为"
- → 双方都获得**相生加成**：木的生长速度+20%，水的共鸣加速+20%
- → 加成持续7天，需要持续"接"才能维持

> 设计意图：让不同属性的创造者**有动力主动找彼此合作**，而不是各做各的。

---

### 五行怎么"长出来"？

系统通过三个维度判断你的主导力量——
1. **你发布了什么类型的内容**（产品/内容/服务/活动）
2. **别人因为什么和你产生碰撞**（买了你的产品 / 被你的内容打动 / 委托你做了服务）
3. **你的收益主要来自哪个方向**（产品销售 / 内容传播 / 服务委托 / 活动组织）

三个月后，你的灵魂画像里会有一朵**五瓣花**，每片花瓣的大小代表你和那种力量的共振强度。

你不是"火型创造者"，你是"与火共振的创造者"。

### 开拓者是什么？

开拓者是拓圈里的"老玩家"——你在拓圈里积累了足够的等级、完成了足够多的碰撞和创作，你觉得这个平台不只是"用来卖东西的"，而是值得你花额外精力去经营的一个地方。

开拓者不是"管理员"，不是"版主"，是**认领了一块责任田的人**。

### 如何申请成为开拓者？

1. 你的创造者等级达到 **Lv.5 以上**
2. 在拓圈内完成了至少 **5 笔有效交易或委托**
3. 你的灵魂画像五瓣花里，至少有一瓣达到 **"熟练"级别**
4. 向拓圈提交"开拓者申请"——写一段你想为这个空间做什么
5. **开荒者（菜菜）审批通过**

### 开拓者可以做什么？

1. **建立运营星球**：你可以在拓圈里建立自己的运营星球（部落升级版），聚集一批和你同频的人，你是这个空间的主理人
2. **发起活动**：你可以发起拓圈官方认证的活动（市集/沙龙/工作坊），吸引更多人进来碰撞
3. **参与平台共建**：拓圈的新功能，开拓者可以优先参与测试和共创，你的建议会直接影响产品方向
4. **收益分配加成**：开拓者的收益分配权重比普通创造者高 **+20%**

### 开拓者的责任

开拓者不是"官职"，是**一种选择**。

你选择了为这个空间多付出一些， correspondingly，你也获得更多的机会和回报。

如果你不想做了，可以随时退出开拓者身份——**不影响你的创造者身份，不影响你的号段资产，保证金照退**。

> 拓圈不是"平台雇佣你"，是"你愿意的话，一起把这个地方做得更好"。

<summary><strong>七、你可能想问的（FAQ）</strong></summary>

**Q1：创造者和探索者可以同时是吗？**
可以。很多人是先成为探索者，在拓圈里碰撞了一段时间、做了一些东西之后，再申请成为创造者的。你的探索者等级、号段、灵魂画像，全部保留，带入创造者身份。

**Q2：保证金什么时候退？**
如果不想继续做了，可以随时申请退保证金。
**如果有用户已经购买你的产品或委托你服务，需要完成所有已产生的交付后再退出。**
没有投诉记录、账号无违规，审核通过后7个工作日内退回。

**Q2补充：我的号段值钱吗？**
非常值钱。你的号段 = 等级积累 + 产品资产 + 专业档案 + 灵魂画像完整度 + 五行属性成长记录。**这个号未来可以交易**——越早入场，积累越多，号段价值越高。你经营的不只是一个创造者资格，是一个会增值的资产。

**Q3：我的产品要在哪里上架？**
产品在拓圈H5里展示，交易在微信小程序里完成。两个地方互相跳转，用户体验是连贯的。

**Q4：我不是"小而美"，我是大品牌，能进来吗？**
拓圈的核心是为"小而美"服务的。大品牌如果想进来，需要接受一个条件：不以"大品牌"的身份出现，而是以"在这个平台上的一个小角色"的身份出现。不造神，不论大小。

**Q5：五行属性会影响我的曝光吗？**
会，但不是"火型一定比水型曝光多"这种简单逻辑。五行影响的是**你被什么类型的人碰撞到的概率**。比如水型创造者的内容更容易被"看重内容质量"的探索者看到。本质上，五行是让"对的人"更容易碰到一起。

**Q6：收益分配大概能拿多少？**
和探索者一样，取决于两个东西：你的等级档位，以及这个档位里有多少人。创造者因为有多一个"产品/服务被点亮"的维度，分配权重通常比同等级探索者高。但具体金额没办法承诺，因为拓圈整体的收益本身也在成长中。

**Q7：我可以不做产品，只做内容吗？**
可以。内容型创造者就是专门做内容的，不需要有实体产品。你的"产品"就是你的内容，别人点亮你的内容，你获得影响力，也可以接委托（比如品牌找你合作）。

**Q8：委托接单会不会很多，忙不过来？**
系统不会给你强推委托。你自己的空间你做主，接不接你自己选。你也可以设置"暂停接单"，这段时间就不会收到新的委托通知。
