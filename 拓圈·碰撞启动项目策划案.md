# 拓圈 · 碰撞启动项目策划案

> **项目代号**：拓圈启动器
> **版本**：V1.0
> **日期**：2026年4月17日
> **定位**：拓圈从0到1的获客引擎 + 内容引擎 + 信任引擎，三位一体

---

## 一、碰撞理念

### 1.1 什么是碰撞

每个人都在自己的轨道上跑。

平时跟人聊的，都是表面的、客套的、功能性的——"最近怎么样""在忙什么""有机会合作"。没有人真正问你"你最想做的那件事是什么"，更没有人认真听完。

**碰撞，就是创造一个没有任何功利目的的空间。**

我不卖你东西，你不防备我。我们就是认认真真碰一次。

### 1.2 为什么叫"碰撞"

碰撞的物理意义：两个物体接触，释放能量。

人和人之间也一样。日常对话是"擦过"——各说各的，礼貌点头，说完就忘。

**碰撞是"迎面撞上"。**

不是谁说服谁，不是谁教育谁，是两套思维、两种经历、两个世界正面相遇。

撞出来的那个东西，叫**火花**——

一句以前没人说过的话，
一个以前没人想过的角度，
一种以前没人体会过的理解。

火花不属于任何一方，它是**碰撞本身产生的**。这就是碰撞最迷人的地方：你不知道会撞出什么，但你确信一定会有。

### 1.3 碰撞的三个原则

**① 不预设，不引导，不强求**

没有台本，不预设问题，不强求"必须聊出什么"。每次碰撞都是独一无二的，接受它本来的样子。

**② 认真听，真实回应**

不是"嗯嗯好的然后呢"式的假装倾听，是真的听进去、接住、给出真实反馈。对方能感觉到你是不是在敷衍。

**③ 到点就断，留下余味**

时间到，立刻停。不拖泥带水，不"再聊五分钟"。正因为时间有限，双方才会珍惜每一秒。而"没聊完"的那个感觉，恰恰是下次连接的理由。

### 1.4 一句话定义

> 我跟你碰一次，碰出火花，你带走一句真话，我带走一个故事。

## 二、项目本质

### 2.1 核心洞察

拓圈冷启动最大的问题不是"没资源"，而是"没有温度"。

直接讲拓圈 = 讲概念 = 对方觉得你在推销 = 防御心理上线 = 没有任何信息能进去。

**真正的信任不是从"我告诉你我是谁"开始的，而是从"我认真听你说你是谁"开始的。**

碰撞就是建立信任的第一步。不是"我先了解你然后再决定要不要推拓圈给你"，而是**"碰撞本身就是价值，拓圈是碰撞的自然延伸"**。

### 2.2 为什么这件事能成飞轮

```
碰撞 → 产出金句/故事 → 传播吸引更多人 → 更多碰撞 → 更多内容 → 更大引力
```

- **对菜菜**：每次碰撞都是一次深度了解潜在创造者的机会，自带筛选功能
- **对被碰者**：被认真对待的感觉 + 拿到一句金句/一个洞察，有价值感
- **对旁观者**：看到真实的人说真实的话，被吸引，想被碰

## 二、三个候选名称

### 2.1 名称分层逻辑

| 层级 | 名称 | 角色 | 使用场景 |
|------|------|------|---------|
| 品牌活动名 | **不废话局** | 正式对外名称，有辨识度和记忆点 | 海报、活动页、传播物料 |
| 邀请话术 | **叮！你被Topoo了** | 发出邀请时的开场白/卡片标题 | 微信私聊、邀请卡片、朋友圈 |
| 轻量互动名 | **拓圈得得~** | 轻松、口语化的备选名，降低门槛 | 随性邀请、内部称呼、社群互动 |

### 2.2 名称详解

#### ① 不废话局

- **调性**：态度鲜明、规则感强、有仪式感
- **记忆点**："局"字有牌局/棋局的博弈感，"不废话"自带筛选——来的人都是认真的人
- **Slogan**：废话留给别人，真话留在这里。
- **编号体系**：不废话局 #001、#002……像藏品一样有积累感
- **适合场景**：正式邀请、海报传播、金句排行榜

#### ② 叮！你被Topoo了

- **调性**：游戏化通知感、惊喜感、排他感
- **记忆点**：模仿游戏/APP推送通知，天然引起好奇——"发生了什么？"
- **Slogan**：你被选中了，现在你有20分钟。
- **卡片形式**：可做成"TOPOO通行证"视觉卡片，微信发给对方
- **适合场景**：一对一邀请、触达话术、裂变传播

#### ③ 拓圈得得~

- **调性**：轻松、口语化、亲切无压力
- **记忆点**："得得"像游戏得分/获得道具的音效（dīng dīng / dé dé），也像两个人碰杯的声音
- **Slogan**：来，碰一下。
- **适合场景**：随性聊天时的邀约、社群互动、内部玩法

### 2.3 使用策略

```
第一步（邀请）：微信发给对方 → "叮！你被Topoo了"（卡片+简短话术）
第二步（参与）：进入碰撞 → "欢迎来到不废话局 #0XX"（仪式感开场）
第三步（传播）：碰撞结束 → 金句海报署名"不废话局出品"（品牌沉淀）
第四步（日常）：社群/朋友圈 → "拓圈得得~ 新一期碰完啦"（轻量触达）
```

## 三、核心机制设计

### 3.1 碰撞规则（不废话局的"游戏规则"）

| 规则 | 说明 |
|------|------|
| **王者局时间** | 默认20分钟一局，跟王者荣耀排位赛一样长，双方各10分钟，你打完我打，节奏感拉满 |
| **10+10对半分** | 前10分钟：对方主场，菜菜认真听、接住、追问；后10分钟：菜菜主场，回应、碰撞、出火花。双方都有完整的表达空间 |
| **到点即断** | 时间一到，无论聊到哪，立刻结束。这是核心规则，不可违反 |
| **没有寒暄** | 不问"最近怎么样"，不上客套，直接进入正题 |
| **一句话收尾** | 到点前1分钟提醒，最后必须用一句话总结今天的碰撞 |
| **全程记录** | 征得同意后，记录关键内容，碰撞后提炼金句 |
| **金句署名** | 产出的金句/洞察，署名"不废话局 #0XX · XXX说" |

**为什么叫"王者局"？**
- 王者荣耀一局排位15-20分钟，这个时长已经被亿万人验证过——足够打出节奏、体验完整、不会疲劳
- "王者局"天然有竞技感和仪式感，被邀进王者局的人心态上就会认真对待
- 10+10的对称结构像游戏里的攻守轮换——你先手我后手，双方都有高光时刻

### 3.2 碰撞流程（像游戏关卡一样清晰）

```
┌─────────────┐
│  ① 发出邀请  │  "叮！你被Topoo了"+ 通行证卡片
└──────┬──────┘
       ▼
┌─────────────┐
│  ② 确认时间  │  对方选择可接受时段 + 时长
└──────┬──────┘
       ▼
┌─────────────┐
│  ③ 开局      │  "欢迎来到不废话局 #0XX，王者局，20分钟，各10分钟"
└──────┬──────┘
       ▼
┌─────────────┐
│  ④ 对方主场  │  前10分钟——对方说，菜菜听、接住、追问
│  （你先手）   │  "这是你的10分钟，随便聊"
└──────┬──────┘
       ▼
┌─────────────┐
│  ⑤ 换手提醒  │  "10分钟到，换我"
└──────┬──────┘
       ▼
┌─────────────┐
│  ⑥ 菜菜主场  │  后10分钟——菜菜回应、碰撞、追问、出火花
│  （我后手）   │  "刚才你说的XXX，我想碰一下……"
└──────┬──────┘
       ▼
┌─────────────┐
│  ⑦ 1分钟提醒 │  "最后1分钟，准备你的收句话"
└──────┬──────┘
       ▼
┌─────────────┐
│  ⑧ 收句话    │  双方各用一句话总结今天的碰撞
└──────┬──────┘
       ▼
┌─────────────┐
│  ⑨ 产出交付  │  金句海报 + 碰撞笔记 → 发给对方
└──────┬──────┘
       ▼
┌─────────────┐
│  ⑩ 传播裂变  │  对方愿意转发 → 吸引下一个人被Topoo
└─────────────┘
```

### 3.3 碰撞话术（参考，不是台本）

**开局话术：**
- "欢迎来到不废话局 #0XX。王者局，20分钟，各10分钟。前10分钟是你的，随便说，我听。后10分钟是我的，我跟你碰。到点就断，不废话。准备好了吗？开始。"

**对方主场阶段（前10分钟）——菜菜的角色：听+接+追问**
- 只听，不打断，不做评价
- 追问示例："然后呢？""为什么是这个方向？""最难的是哪一步？"
- 如果对方卡住："没关系，想到什么说什么，这里没有错误答案。"

**换手话术（第10分钟）：**
- "10分钟到，换我。"
- "你的时间到了，现在轮到我出招。"
- "OK你先手打完了，我后手来了——刚才你说的XXX，我想碰一下。"

**菜菜主场阶段（后10分钟）——菜菜的角色：回应+碰撞+出火花**
- 回应对方说过的点："你刚才说的XXX，我有一个不同的看法……"
- 碰撞：不是反驳，是从另一个角度切入同一个问题
- 可以分享自己的经历/观察，但核心还是围绕对方
- 追问更深的东西："如果没有人看，你还会做吗？""你最想改变什么？"

**最后1分钟提醒：**
- "最后1分钟。准备你的收句话——如果今天这次碰撞只能留下一句话，你说什么？"

**收尾话术：**
- "时间到。不废话局 #0XX，结束。你的收句话我记下了，回头发你金句海报。"
- "没聊完的，下次再碰。"

## 四、产出物体系

### 4.1 单次碰撞产出

| 产出物 | 形式 | 用途 |
|--------|------|------|
| **金句海报** | 图文海报，深紫风格，署名编号 | 对方发朋友圈/小红书，传播裂变 |

| **收句话** | 一句话，加粗突出 | 所有产出的核心，最易传播 |

## 五、裂变机制设计

### 5.1 一级裂变：被碰者传播

- 碰撞结束后，金句海报发给对方
- 对方发朋友圈 = 一次免费传播
- 海报底部带："想被Topoo？联系XXX" 或 小程序码

### 5.2 二级裂变：围观者转化

- 看到金句海报的人 → 好奇 → 想被碰 → 主动联系
- 可以做"提名机制"——A被碰完，可以提名B："我想看XXX被Topoo"

### 5.3 三级裂变：内容矩阵

- 金句合集 → 公众号推文
- 碰撞片段 → 抖音/小红书短视频
- 月度精选 → 小程序专题页
- 每一条内容都是下一次碰撞的广告

### 5.4 轻量标识（噱头，不是主体）

碰撞的核心价值是"言之有物"，所有外在形式只是让人记住、愿意来的钩子。

| 元素 | 说明 |
|------|------|
| **编号** | 不废话局 #001 起，永不重复 |
| **限量感** | 种子期限定XX局，制造稀缺 |
| **提名** | 被碰过的人可以提名下一个人 |

不搞段位、不搞成就、不搞排行。碰撞本身就是最好的内容，金句本身就是最好的传播。

## 六、菜菜的角色定位

### 6.1 不是什么

- 不是主持人（没有台本）
- 不是销售（不推销拓圈）
- 不是采访者（不预设问题）
- 不是导师（不说教）

### 6.2 是什么

- **一个认真的倾听者**：真心对对方的故事感兴趣
- **一个有阅历的对话者**：能接住对方的话，给出真实反馈
- **一个有商业直觉的人**：聊的过程中自然判断——这个人适合拓圈的什么角色
- **一个内容提炼者**：碰撞完能提炼出最有价值的那句话

### 6.3 核心原则

> **碰撞的目的是让对方觉得"被认真对待了"，而不是让对方觉得"被营销了"。**

拓圈的名字在碰撞过程中**可以完全不提**，或者只在最后自然带一句："我在做一个叫拓圈的事，可能跟你有关，有兴趣的话可以看看。"

## 七、执行计划

### 7.1 第一阶段：冷启动（第1-4周）

| 周次 | 目标 | 动作 |
|------|------|------|
| 第1周 | 跑通模型 | 碰3-5个信任度高的人（朋友/熟人），测试流程和话术 |
| 第2周 | 优化体验 | 根据反馈调整时长、话题引导、产出物模板 |
| 第3周 | 开始裂变 | 碰5-8个弱关系人，正式启动金句海报传播 |
| 第4周 | 复盘迭代 | 统计裂变数据，优化邀请话术和海报设计 |

### 7.2 第二阶段：规模化（第5-12周）

| 周次 | 目标 | 动作 |
|------|------|------|
| 第5-6周 | 建立节奏 | 固定每周碰3-5人，形成内容产出节奏 |
| 第7-8周 | 内容分发 | 公众号/小红书/抖音同步推送金句内容 |
| 第9-10周 | 提名机制 | 上线"我提名XXX被Topoo"功能 |
| 第11-12周 | 阶段复盘 | 统计转化率（被碰→了解拓圈→入驻），优化漏斗 |

### 7.3 第三阶段：品牌化（第3个月起）

- 不废话局成为拓圈的标志性IP活动
- 可以开放"分局"——其他城市/其他人的不废话局
- 年度碰撞盛典（线下活动，把被碰过的人聚在一起）

## 八、关键指标

| 指标 | 定义 | 目标（第一阶段） |
|------|------|-----------------|
| 碰撞总数 | 累计完成的不废话局次数 | 20-30局 |
| 裂变率 | 被碰者发朋友圈的比例 | ≥60% |
| 提名率 | 被碰者提名下一个人的比例 | ≥40% |
| 金句传播量 | 单条金句海报的平均曝光 | ≥200次 |
| 转化率 | 被碰者→主动了解拓圈 | ≥30% |
| 入驻率 | 被碰者→正式入驻拓圈 | ≥10% |

## 九、风险与应对

| 风险 | 应对策略 |
|------|---------|
| 对方觉得是营销 | 碰撞过程中不提拓圈，只在最后自然带一句 |
| 金句质量不稳定 | 不是每次都有好金句，不强求，有就发，没有就只留存内部笔记 |
| 时间控制不住 | 用计时器，到点硬断，形成"不废话"的品牌记忆 |
| 对方不愿意传播 | 不强迫，金句海报是送给对方的，发不发随他 |
| 碰撞对象不够精准 | 从文创/设计/内容创作者开始，围绕拓圈目标人群 |
